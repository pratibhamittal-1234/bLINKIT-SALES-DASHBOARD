# 📊 Blinkit Sales Dashboard | Power BI

## 📌 Project Overview

This project is an interactive Power BI dashboard built to analyze Blinkit's sales performance and provide actionable business insights. The dashboard presents key performance indicators (KPIs), sales trends, outlet analysis, and product performance through interactive visualizations.

---

## 🎯 Objective

The objective of this project is to transform raw sales data into meaningful insights that help businesses:

- Monitor overall sales performance
- Analyze product category performance
- Compare outlet sales across different locations
- Identify customer purchasing patterns
- Support data-driven business decisions

---

## 🛠️ Tools & Technologies Used

- Power BI
- Power Query
- DAX
- Microsoft Excel
- Data Visualization

---

## 📈 Dashboard Features

- Total Sales KPI
- Average Sales
- Number of Items Sold
- Average Customer Rating
- Sales by Item Type
- Sales by Outlet Size
- Sales by Outlet Location
- Fat Content Analysis
- Interactive Filters & Slicers

---

## 💡 Key Insights

- Identifies top-performing product categories.
- Compares sales across different outlet locations.
- Helps understand customer purchasing trends.
- Supports inventory and marketing decisions.
